# Telecom-customer-churn-prediction
Customer churn prediction using ML and Deep Learning (Python, Pandas, Scikit-learn, Decision Tree, SMOTE, SHAP)

## Problem
Predict whether a customer will churn or not.

## Tech stack
Python, Pandas, Numpy, Scikit-learn, TensorFlow, Keras

## Models
Logistic Regression, Decision Tree, Random Forest, KNN, SVM, Naive Bayes, ANN(Deep Learning)

## Techniques
Stratified Split, Smote, SHAP

## Evaluation
Accuracy, Confusion Matrix, ROC-AUC

## Result
Achieved ~81% accuracy
